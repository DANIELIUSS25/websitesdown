# WebsiteDown.com

Check if any website is down — instantly.

## Deploy

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/DANIELIUSS25/websitedown1)

## Stack

- Static HTML / CSS / JS
- Deployed on Netlify
- Zero build step

## Local Development

```bash
# Just open the file
open public/index.html
```

## Structure

```
public/
  index.html    ← entire homepage
netlify.toml    ← deploy config + redirects
```
